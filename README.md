# QuantumNestedSampling

Julia library specializing nested sampling and Galilean Monte Carlo to quantum states on finite-dimensional Hilbert spaces (esp. spin systems).

NB this is research software, released primarily for the sake of reproducibility of an upcoming paper. You're welcome to use it, but beware. If you do use it, please let me know how—I'm curious!
